# Temple Scan Learn Discover (AR Application)
Temple Scan Learn Discover is an augmented reality (AR) application that allows users to scan temple elements and explore interactive information in real time. The application enhances learning through immersive visualization.

## Tech Stack
- Unity
- Vuforia SDK

## How It Works
1. Scan a temple image/object using the app
2. Vuforia detects the target
3. AR content is displayed
